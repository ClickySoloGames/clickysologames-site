ClickySoloGames

Independent browser game developer focused on lightweight, fast-loading HTML5 games playable directly in modern browsers.

Projects are developed using Unity WebGL and modern browser technologies with focus on performance and cross-platform compatibility.

Games run smoothly across different screen resolutions and devices.

Car Parking

Casual browser puzzle game built with Unity WebGL

ClickySoloGames Project

Car Parking is a lightweight puzzle-style browser game where players need to move vehicles in the correct order to clear the parking area.

The gameplay focuses on logical thinking and short casual sessions.
