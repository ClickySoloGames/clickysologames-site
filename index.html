ClickySoloGames

ClickySoloGames is an independent browser game developer focused on creating lightweight, fast-loading and accessible HTML5 games playable directly in modern browsers.

The goal is to design simple and engaging gameplay experiences optimized for both desktop and mobile devices without requiring installation.

Technology

Projects are developed using Unity WebGL and modern browser technologies with strong focus on performance optimization and cross-platform compatibility.

Games are designed to run smoothly across different screen resolutions and devices.

Car Parking

Casual browser puzzle game built with Unity WebGL

ClickySoloGames Project

Car Parking is a lightweight puzzle-style browser game where players need to move vehicles in the correct order to clear the parking area.

The gameplay focuses on logical thinking and short casual sessions.
